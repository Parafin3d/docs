---
title: "Navigating Parafin"
description: "Use the main menu, user menu, and Project Panel to move between projects, rename, and access account options."
---

## Main menu

The **main menu dropdown** in the top-left corner takes you back to My Projects and other common destinations from anywhere in Parafin.

  <img src="/images/releases/1.57/main_menu.gif" alt="Opening the main menu dropdown from the top-left corner" />
</Frame>

## Rename a project

Click the project name at the top of the **Project Panel** to edit it inline. Changes save automatically.

<Frame caption="Editing the project name directly in the Project Panel">
  <img src="/images/releases/1.54/edit_project_name.png" alt="Inline editing of the project name in the Project Panel" />
</Frame>

## User menu

The **user menu** in the top-right corner opens account settings, password changes, and sign-out.
