The first step asks you to give your project a name. The cursor starts in the text field so you can begin typing immediately.

<Frame caption="Give your project a name">
  <img src="/images/help/launching/project-name.png" alt="Project name step showing text field with Type here placeholder" />
</Frame>

A note below the field confirms that the project name can be changed at any time. Type a name and click **Next** or press **Enter** to proceed to the Location step.
