The Zoning Panel brings every available zoning detail for your site into a single, easy-to-navigate location. Instead of juggling browser tabs and PDFs from municipal websites, you can review district codes, dimensional controls, and direct links to the source — all from inside your project.

This guide walks you through opening the panel, reading what's inside, and jumping to the underlying municipal zoning code.

## Open the Zoning Panel

<Steps>
  <Step title="Open a project with a defined site">
    Start from any project where you've already defined a site location. The panel needs a parcel to query against.
  </Step>
  <Step title="Click into project information">
    Click anywhere in the **project information section** at the top of the screen.
  </Step>
  <Step title="Expand the Zoning section">
    Expand the **Zoning** section. The panel slides open with all available zoning data for your selected parcel.
  </Step>
</Steps>

<Note>
  If your parcel doesn't show zoning data, your jurisdiction may not yet be in the Zoneomics feed. [Let us know](mailto:support@parafin.ai) and we'll add it in the next weekly update.
</Note>

## What's in the panel

Inside the panel you'll see:

| Field | What it tells you |
| --- | --- |
| **District code** | The official zoning designation for your parcel (e.g., R-3, C-1, M-2) |
| **Description** | A plain-language summary of what the district allows |
| **Zoning type** | The category — residential, commercial, mixed-use, etc. |
| **Zoning code link** | A direct link to the municipal source for the full code |
| **Dimensional controls** | The key constraints that drive feasibility: FAR, max building height, minimum open space, and setback requirements |

## Open the source code

Click the **Zoning Code Link** to open the full municipal code for your parcel in a new tab. This takes you straight to the relevant section of the city or county's published zoning ordinance — no manual searching required.

<Tip>
  Bookmark frequently-used jurisdictions in your browser. The code links are stable and update with the parcel data.
</Tip>

## Applying zoning constraints to your project

The Zoning Panel exposes a set of **zoning filters** for FAR, maximum building height, and minimum open space. Each constraint displays a suggested zoning value — apply it as-is, enter your own value using the text field or slider, or turn the control off for that metric. Changes update your project assumptions in real time, and variant generations adjust accordingly.
