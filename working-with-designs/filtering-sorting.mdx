---
title: "Filtering and sorting designs"
description: "Narrow the design set and sort by financial metrics."
---

Parafin generates thousands of candidate designs per site. Use filters to narrow the set and sort controls to surface top options by the metrics that matter to you.

## Filters

Set boundaries on:

- FAR
- Building height
- Number of keys
- Room mix
- Program elements

</Frame>

## Sort

Sort the design list by any financial metric:

- Number of keys
- Budget per key
- Equity multiple
- Yield

<Frame>
  <img src="/images/support/legacy/designs/sorting-designs.jpg" alt="Sort controls and sorted design list" />
</Frame>

## Star and hide

Click the star icon on any design to pin it to the top of the list regardless of sort. Click the **X** to hide a design from the list entirely.
