Six steps take you from a site address to a full set of generated designs. Parafin handles the rest.

## The steps

<CardGroup cols={2}>
  <Card title="1. Project name" href="/launching-a-project/project-name">
    Give the project a unique, descriptive name.
  </Card>
  <Card title="2. Site address" href="/launching-a-project/site-address">
    Enter a U.S. address to locate the site.
  </Card>
  <Card title="3. Site area" href="/launching-a-project/site-area">
    Select parcels on the map to define the site.
  </Card>
  <Card title="4. Draw site area" href="/launching-a-project/drawing-site-area">
    Or draw a custom boundary by hand.
  </Card>
  <Card title="5. Setbacks and adjacency" href="/launching-a-project/setbacks-adjacency">
    Set adjacency rules and required setbacks.
  </Card>
  <Card title="6. Choose a brand" href="/launching-a-project/hotel-brand">
    Pick a hotel brand and run Parafin.
  </Card>
</CardGroup>

## Start a new project

Start a new project from any of three entry points:

- The **New Project** button on your dashboard
- **Add a New Project** link on the dashboard
- Direct URL: [app.parafin.ai/dashboard/create](https://app.parafin.ai/dashboard/create)
